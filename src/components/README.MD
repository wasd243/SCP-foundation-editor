## In `conponents\` `conponents\editor\`

This folder contains all the context components.

| Name       | Description                                                                  |
|------------|------------------------------------------------------------------------------|
| `editor`   | Editor buttons, canvas and ribbon pages for editor components                |
| `settings` | The editor's settings panel which includes setting buttons, dev output, etc. |

All `.vue` files which about editor buttons for editor components should be placed in this folder.
