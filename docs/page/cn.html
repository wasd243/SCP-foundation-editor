已知限制

为何 [[include]] 无法所见即所得。

[[include]] 看似 C 语言的 #include，但它并非文本替换——而是带变量插值与静默回退的参数化宏。

宏展开是有损的，嵌套时更不可逆：展开之后，模板内容、传入参数与作者文本已无法区分——因此没有任何正确的位置可以把编辑写回。包含块只能单向工作。

技术栈

它由什么构成。

Tauri 2 跨平台桌面外壳。
Vue 3 + TS 编辑器界面。
TipTap / ProseMirror 所见即所得的核心。
ftml 第三方 Wikidot → HTML 解析器，由 SCP Wiki 团队开发。 github.com/scpwiki/ftml ↗
Rust 解析管线、导出器与 IPC。
SCP-WYSIWYG

用于撰写 SCP Wiki 文章的社区工具。Wikidot 解析由 ftml 驱动——它是由 SCP Wiki 团队维护的独立库。主题素材以 CC BY-SA 3.0 授权。
